This application is designed for working with independent Checkbutton controls. Unlike radio buttons, checkboxes allow multiple options to be selected simultaneously. 
Each control is bound to its own Boolean variable, tk.BooleanVar(). When the "Calculate Total" button is clicked, the program checks the state of each checkbox using 
the .get() method. If the checkbox is active (True), the corresponding cost is summed up, and the option name is added to the resulting array displayed on the screen. 
The "Reset" button resets all Boolean variables to their initial state of False.
